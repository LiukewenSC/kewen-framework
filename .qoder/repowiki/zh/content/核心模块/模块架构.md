# 核心模块架构

## 模块总览

Kewen Framework 采用 Maven 多模块架构，各模块职责清晰，按需引入。

## 模块依赖关系

```
kewen-framework (父工程)
├── basic (基础模块)
│   └── basic-support (基础支持)
├── boot (Spring Boot Starter)
│   ├── basic-spring-boot-starter
│   ├── storage-spring-boot-starter
│   ├── tenant-spring-boot-starter
│   └── magic-spring-boot-starter
├── qy-auth (权限认证)
│   ├── auth-core
│   ├── auth-core-spring-boot-starter
│   ├── auth-rbac
│   ├── auth-rbac-spring-boot-starter
│   └── auth-spring-boot-starter
├── qy-idaas (IDaaS认证)
│   └── idaas-authentications
├── sample (示例项目)
└── other (工具)
    └── code-generator
```

## 基础模块 (basic)

### 功能说明

提供通用的基础功能，是所有其他模块的依赖基础。

### 包结构

```
com.kewen.framework.basic
├── exception/          # 异常处理
│   ├── BusinessException      # 业务异常
│   ├── SystemException        # 系统异常
│   └── ...
├── filter/             # 过滤器
├── logger/             # 日志管理
│   ├── RequestLogger          # 请求日志
│   └── ...
├── model/              # 通用模型
│   ├── Result                 # 统一返回结果
│   ├── PageReq                # 分页请求
│   └── ...
└── utils/              # 工具类
    ├── DateUtils
    ├── StringUtils
    └── ...
```

### 核心类

#### Result 统一返回

```java
public class Result<T> {
    private int code;
    private String message;
    private T data;
    
    public static <T> Result<T> success(T data) { }
    public static Result<?> failed(int code, String message) { }
}
```

## Spring Boot Starter 模块 (boot)

### 模块说明

提供即插即用的自动配置功能。

### basic-spring-boot-starter

**功能：**
- 请求日志记录
- 请求跟踪（TraceId）
- 方糖消息通知
- 数据库请求持久化

**配置：**
```yaml
kewen:
  request:
    message:
      fang-tang: true
    persistent:
      database: true
```

### storage-spring-boot-starter

**功能：**
- 文件上传下载
- 多存储平台支持（本地、七牛云等）
- 分片上传
- 缩略图生成

**核心表：**
- `file_detail` - 文件记录表
- `file_part_detail` - 文件分片信息表

**配置：**
```yaml
kewen:
  storage:
    platform: qiniu  # 存储平台
    qiniu:
      access-key: xxx
      secret-key: xxx
      bucket: xxx
      domain: xxx
```

### tenant-spring-boot-starter

**功能：**
- 多租户架构支持
- 租户上下文管理
- 数据隔离

**使用：**
```java
// 设置租户
TenantContext.setTenantId("tenant001");

// 获取租户
String tenantId = TenantContext.getTenantId();
```

**配置：**
```yaml
kewen:
  tenant:
    open: true  # 是否开启多租户
```

### magic-spring-boot-starter

**功能：**
- Magic API 集成
- 低代码 API 开发

## 权限认证模块 (qy-auth)

### 模块结构

```
qy-auth/
├── auth-core/                          # 权限核心
│   ├── annotation/                     # 注解
│   │   ├── AuthMenu                   # 菜单权限
│   │   ├── AuthDataRange              # 数据范围
│   │   ├── AuthDataOperation          # 数据操作
│   │   └── AuthDataAuthEdit           # 数据编辑
│   ├── model/                          # 模型
│   │   ├── BaseAuth                   # 基础权限
│   │   ├── IAuthEntity                # 权限实体接口
│   │   └── IAuthObject                # 权限集合接口
│   └── extension/                      # 扩展
│       ├── AbstractAuthObject
│       └── AbstractIdNameFlagAuthEntity
├── auth-core-spring-boot-starter/      # 权限核心配置
├── auth-rbac/                          # RBAC实现
│   ├── controller/                     # 控制器
│   │   ├── RabcUserController
│   │   ├── RabcRoleController
│   │   ├── RabcDeptController
│   │   ├── RabcMenuApiController
│   │   ├── RabcMenuRouteController
│   │   └── RabcCurrentUserController
│   ├── model/                          # 数据模型
│   ├── mp/                             # 持久层
│   ├── composite/                      # 组合服务
│   └── utils/                          # 工具类
├── auth-rbac-spring-boot-starter/      # RBAC配置
└── auth-spring-boot-starter/           # Security配置
    ├── filter/                         # 过滤器
    │   ├── JsonLoginFilter
    │   ├── TokenSessionRequestFilter
    │   └── AuthUserContextFilter
    ├── config/                         # 配置
    ├── service/                        # 服务
    └── model/                          # 模型
```

### 模块职责

#### auth-core

- 定义权限注解
- 定义权限模型
- 提供权限处理接口
- 不依赖具体实现

#### auth-rbac

- 实现 RBAC 权限体系
- 提供用户、角色、部门管理
- 实现权限验证逻辑
- 提供权限管理 API

#### auth-spring-boot-starter

- Spring Security 配置
- 登录认证流程
- Token 管理
- Session 管理

## IDaaS 模块 (qy-idaas)

### 功能

提供企业级身份认证服务集成：
- SAML 认证
- OAuth 认证
- 单点登录（SSO）

### 模块结构

```
qy-idaas/
└── idaas-authentications/
    └── SAML/OAuth 认证实现
```

## 示例项目 (sample)

### 示例列表

1. **auth-boot-sample**
   - 完整的权限认证示例
   - 包含 RBAC 使用
   - 展示权限注解使用

2. **basic-boot-sample**
   - 基础功能示例
   - 请求日志演示
   - 生命周期监听器

3. **storage-boot-sample**
   - 文件存储示例
   - 上传下载演示

4. **tenant-boot-sample**
   - 多租户示例
   - 租户切换演示

5. **idaas-sp-boot-sample**
   - IDaaS SAML 示例
   - SSO 集成演示

## 工具模块 (other)

### code-generator

**功能：**
- MyBatis-Plus 代码生成
- 自动生成 Entity、Mapper、Service、Controller

**使用：**
```java
MybatisPlusCodeGenerator generator = new MybatisPlusCodeGenerator();
generator.generate("表名");
```

## 模块引入指南

### 最小权限系统

```xml
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-spring-boot-starter</artifactId>
</dependency>
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-rbac-spring-boot-starter</artifactId>
</dependency>
```

### 完整功能系统

```xml
<dependency>
    <groupId>com.kewen.framework.boot</groupId>
    <artifactId>basic-spring-boot-starter</artifactId>
</dependency>
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-spring-boot-starter</artifactId>
</dependency>
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-rbac-spring-boot-starter</artifactId>
</dependency>
<dependency>
    <groupId>com.kewen.framework.boot</groupId>
    <artifactId>storage-spring-boot-starter</artifactId>
</dependency>
```

### 多租户系统

```xml
<dependency>
    <groupId>com.kewen.framework.boot</groupId>
    <artifactId>tenant-spring-boot-starter</artifactId>
</dependency>
```

## 版本管理

### 依赖版本

- Spring Boot: 2.7.18
- Spring Cloud: 2021.0.9
- MyBatis-Plus: 3.5.8
- Hutool: 5.8.18
- FastJSON: 2.0.25
- Lombok: 1.18.30

```xml
<parent>
    <groupId>com.kewen.framework</groupId>
    <artifactId>kewen-framework</artifactId>
    <version>1.0-SNAPSHOT</version>
</parent>
```
