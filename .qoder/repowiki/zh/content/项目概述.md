# Kewen Framework 后端项目

## 项目概述

Kewen Framework 是一个基于 Java + Spring Boot 的企业级后端框架，提供完整的权限管理、数据存储、多租户等企业级功能。

## 核心特性

- **权限管理系统**：基于 RBAC 模型的完整权限控制体系
- **注解驱动**：通过注解实现菜单权限和数据权限控制
- **多模块架构**：Maven 多模块设计，按需引入功能
- **Spring Security 集成**：安全的认证和授权机制
- **MyBatis-Plus 集成**：高效的数据访问层
- **存储模块**：支持多平台文件存储
- **多租户支持**：内置多租户架构支持

## 技术栈

- **核心框架**：Spring Boot 2.7.18, Spring 5
- **安全框架**：Spring Security
- **ORM 框架**：MyBatis-Plus 3.5.8
- **数据库**：MySQL 8.0
- **构建工具**：Maven
- **工具库**：Hutool 5.8.18, FastJSON 2.0, Lombok 1.18.30
- **API 文档**：Swagger 2.9.2

## 快速开始

### 环境要求

- JDK 8+
- Maven 3.6+
- MySQL 5.7+

### 安装框架

```bash
cd kewen-framework
mvn clean install
```

### 运行示例项目

```bash
cd sample/auth-boot-sample
mvn spring-boot:run
```

## 核心模块说明

### 1. 权限认证模块 (qy-auth)

完整的 RBAC 权限管理系统，支持：
- 菜单权限控制
- 数据权限控制
- 注解驱动的权限验证
- Spring Security 集成

详见：[权限控制系统](./权限控制系统/权限系统概述.md)

### 2. Spring Boot Starter 模块 (boot)

提供即插即用的功能模块：
- **basic-spring-boot-starter**：基础功能（日志、请求跟踪等）
- **storage-spring-boot-starter**：文件存储（支持七牛云等平台）
- **tenant-spring-boot-starter**：多租户支持
- **magic-spring-boot-starter**：Magic API 集成

### 3. 基础模块 (basic)

提供通用的工具类和基础功能：
- 异常处理
- 日志管理
- 通用模型
- 工具类库

## 配置说明

### 基础配置

```yaml
server:
  port: 8081

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/your_database
    username: your_username
    password: your_password
    driver-class-name: com.mysql.cj.jdbc.Driver
```

### 权限模块配置

```yaml
kewen:
  auth:
    auth-data-table:
      table-name: sys_auth_data
    cache-auth: false
  security:
    session:
      maximum-sessions: 10
      max-sessions-prevents-login: true
    login:
      password:
        login-url: /login
        username-parameter: username
        password-parameter: password
```

### 请求日志配置

```yaml
kewen:
  request:
    message:
      fang-tang: true  # 是否发送方糖消息
    persistent:
      database: true   # 是否持久化到数据库
```

## 开发指南

### 引入依赖

在你的项目中引入所需的 Starter：

```xml
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-spring-boot-starter</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
<dependency>
    <groupId>com.kewen.framework.auth</groupId>
    <artifactId>auth-rbac-spring-boot-starter</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```

### 使用权限注解

```java
@RestController
@RequestMapping("/api")
public class YourController {
    
    @AuthMenu(name = "查询列表")
    @AuthDataRange(businessFunction = "your_business")
    @GetMapping("/list")
    public Result list() {
        // 业务逻辑
    }
}
```
