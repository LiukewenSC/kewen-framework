### 需要用户自己配置的自定义参数

示例表明的为默认值

```yaml
kewen:
  message:
    fang-tang:
      key: ''
      domain: https://sctapi.ftqq.com
  request:
    message:
      fang-tang: true
    persistent:
      database: true
  tenant:
    open: false
```
